# LoveEatKeepFit Coaching PWA (App)

This is the PWA version of the LoveEatKeepFit coaching app, accessible at `/app`.

- 100% accessible UI
- GDPR-compliant consent
- Guest login (mocked)
- Day-by-day lessons
- Chat (placeholder)
- Homework submission
- Google Form feedback integration

To move to a separate repo, copy the `src/app` folder and update routing as needed.
