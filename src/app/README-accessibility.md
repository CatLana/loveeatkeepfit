# Accessibility Statement

This app is designed to be fully accessible and compliant with WCAG 2.1 AA standards:

- All interactive elements are keyboard accessible
- Sufficient color contrast is maintained
- ARIA labels and roles are used where appropriate
- Semantic HTML is used for structure
- Focus indicators are visible
- Screen reader support is provided

If you encounter any accessibility issues, please contact info@loveeatkeepfit.ie.
